# 📊 TODO API 실측 벤치마크 결과 리포트

이 리포트는 Apple Silicon(M-시리즈) 환경의 Docker 컨테이너에서 각 프레임워크의 최신 안정화 버전을 대상으로 직접 측정된 데이터를 기반으로 작성되었습니다.

## 1. 실측 데이터 요약

| Service | Framework | RPS (처리량) | CPU % | Memory | Img Size |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **php-laravel** | PHP 8.4 (Minimal) | **164.66** | **0.00%** | 12.2 MiB | 150 MB |
| **go-gin** | Gin 1.10.0 | 162.76 | **0.00%** | **9.93 MiB** | **32.3 MB** |
| **node-express** | Express 4.21.2 | 152.22 | 0.03% | 22.07 MiB | 247 MB |
| **java-spring** | Spring Boot 3.4.3 | 143.53 | 0.34% | 139.9 MiB | 324 MB |
| **python-fastapi** | FastAPI 0.115.8 | 142.74 | 0.26% | 48.0 MiB | 346 MB |

> **참고**: RPS 수치가 낮은 이유는 벤치마크 툴이 `curl` 프로세스를 순차적으로 생성하여 테스트했기 때문입니다. 이는 프레임워크의 한계 수치보다는 **단일 요청 처리 효율성 및 오버헤드**를 비교하는 지표로 해석해야 합니다.

## 2. 심층 분석

### 🥇 처리량 및 효율성 1위: PHP 8.4 (Minimal)
- **분석**: PHP 8.4의 엔진 최적화와 단순 라우팅 구현이 결합되어 가장 빠른 응답 속도를 보여주었습니다. 
- **특이사항**: 가벼운 API 구현 시 PHP가 결코 다른 언어에 뒤처지지 않는다는 것을 입증했습니다. 메모리 사용량도 12.2MiB로 매우 준수합니다.

### 🥈 리소스 가성비 1위: Go (Gin)
- **분석**: Go는 9.93MiB라는 **경이로운 메모리 점유율**을 보여주었습니다. 이미지 크기 또한 32.3MB로 다른 프레임워크들의 1/10 수준입니다.
- **특이사항**: 인프라 비용 절감이 최우선인 클라우드 네이티브 환경(K8s 등)에서 최고의 선택임을 다시 한번 확인했습니다.

### 🥉 안정적인 균형: Node.js (Express)
- **분석**: RPS 152.22로 상위권에 속하며, 메모리 사용량(22.07MiB) 또한 합리적입니다.
- **특이사항**: 대중적인 Express 프레임워크가 성능과 리소스 면에서 여전히 매우 경쟁력 있는 "표준"임을 보여줍니다.

### 🛡️ 대규모 시스템의 기반: Java (Spring Boot)
- **분석**: 메모리 점유율이 139.9MiB로 가장 높지만, 이는 JVM이 미리 할당하는 힙(Heap) 영역 때문입니다. RPS는 Python보다 높게 측정되었습니다.
- **특이사항**: 리소스 소모는 크지만, 요청 부하가 늘어날수록 JVM의 JIT 최적화가 강력해져 실제 대규모 운영 환경에서는 더 높은 안정성을 보일 것입니다.

### 🐍 생산성의 강자: Python (FastAPI)
- **분석**: RPS는 142.74로 가장 낮게 측정되었으며, 이미지 크기(346MB)가 가장 큽니다.
- **특이사항**: 리소스 효율성보다는 개발 생산성과 비동기 처리의 편의성에 강점이 있는 프레임워크입니다.

## 3. 최종 결론

1. **가장 가볍고 효율적인 배포가 필요하다면?** 👉 **Go (Gin)**
2. **단순 API의 빠른 응답 속도가 중요하다면?** 👉 **PHP 8.4**
3. **가장 대중적이고 균형 잡힌 선택을 원한다면?** 👉 **Node.js (Express)**
4. **기업용 대규모 확장성이 필요하다면?** 👉 **Java (Spring Boot)**
5. **AI/Data 연동 및 빠른 개발이 필요하다면?** 👉 **Python (FastAPI)**

본 리포트는 벤치마크 툴(`benchmark-tool`)의 실측 데이터를 기반으로 작성되었습니다.
